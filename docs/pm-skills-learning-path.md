## Level 1: Foundation (5 Skills)

### Learning Objectives
- Understand how to articulate problems from user perspective
- Create basic user persona hypotheses
- Write clear, actionable user stories
- Understand customer motivations and jobs-to-be-done
- Communicate product vision effectively

### Skills

| # | Skill | Type | Description |
|---|-------|------|-------------|
| 1 | `problem-statement` | component | Articulate problems from user perspective (who, what, why, how they feel) |
| 2 | `proto-persona` | component | Create assumption-based persona profiles from research and stakeholder input |
| 3 | `user-story` | component | Write user stories with Mike Cohn format + Gherkin acceptance criteria |
| 4 | `jobs-to-be-done` | component | Explore customer motivations (functional, social, emotional jobs + pains + gains) |
| 5 | `press-release` | component | Create visionary press release using Amazon's "Working Backwards" method |

### Expected Deliverables
- 3 problem statements for your product
- 2 proto-personas representing key user segments
- 10 user stories with acceptance criteria
- 1 JTBD canvas for primary customer
- 1 press release for upcoming feature/product

### Time Estimate
**4-6 hours** to complete all foundation skills

## Level 2: Discovery & Research (6 Skills)

### Learning Objectives
- Prepare and conduct effective customer discovery interviews
- Map customer journeys to identify pain points
- Visualize user workflows with story mapping
- Research competitors and market opportunities
- Calculate addressable market size
- Analyze macro-environmental factors

### Skills

| # | Skill | Type | Description |
|---|-------|------|-------------|
| 1 | `discovery-interview-prep` | interactive | Prepare for customer discovery interviews with adaptive questions |
| 2 | `customer-journey-map` | component | Visualize customer interactions across all stages |
| 3 | `user-story-mapping` | component | Create hierarchical maps of user activities and tasks |
| 4 | `company-research` | component | Create comprehensive company profiles from public sources |
| 5 | `tam-sam-som-calculator` | interactive | Calculate Total/Serviceable/Obtainable Market |
| 6 | `pestel-analysis` | component | Analyze Political, Economic, Social, Technological, Environmental, Legal factors |

### Prerequisite
Complete **Level 1** skills first

### Expected Deliverables
- Discovery interview guide for 5 customers
- 1 customer journey map with pain points and opportunities
- 1 user story map (2-dimensional: backbone + activities)
- Competitive analysis for 3 competitors
- TAM/SAM/SOM calculation for your product
- PESTEL analysis for your market

### Time Estimate
**8-12 hours** to complete all discovery skills

### Learning Objectives
- Create compelling product positioning
- Build opportunity solution trees for systematic solution generation
- Frame epics as testable hypotheses
- Write comprehensive PRDs
- Create strategic roadmaps
- Choose appropriate prioritization frameworks
- Run effective product strategy sessions

### Skills

| # | Skill | Type | Description |
|---|-------|------|-------------|
| 1 | `positioning-statement` | component | Geoffrey Moore-style positioning statement |
| 2 | `opportunity-solution-tree` | interactive | Create OST with outcomes, opportunities, solutions |
| 3 | `epic-hypothesis` | component | Frame epics as testable if/then hypotheses |
| 4 | `prd-development` | workflow | Structured PRD creation process |
| 5 | `roadmap-planning` | workflow | Strategic roadmap planning process |
| 6 | `prioritization-advisor` | interactive | Choose right framework based on context |
| 7 | `product-strategy-session` | workflow | Comprehensive strategy workshop process |

### Prerequisites
- Complete **Level 1** skills
- Complete **Level 2** skills recommended

### Expected Deliverables
- Positioning statement for your product
- 1 opportunity solution tree with 5+ solutions
- 5 epic hypotheses with validation approaches
- 1 comprehensive PRD
- 1 quarterly roadmap with prioritization
- Prioritization framework recommendation with rationale

### Time Estimate
**12-16 hours** to complete all strategy skills

## How to Use This Learning Path

### Option 1: Sequential Learning
Start at Level 1 and progress through each level in order. This provides the most comprehensive foundation.

### Option 2: Role-Based Jump Start
Choose a learning journey based on your role and skip to the most relevant level, then fill gaps as needed.

### Option 3: Skill-First Approach
Pick specific skills from any level based on immediate needs, then expand outward.

### Tips for Success
1. **Practice with real work**: Apply each skill to actual PM problems
2. **Get feedback**: Share deliverables with peers or mentors
3. **Iterate**: Refine outputs based on feedback
4. **Connect skills**: See how skills build on each other
5. **Teach others**: Explaining reinforces learning

## Skills by Type

### Component Skills (20)

Templates and artifacts you can use directly.

| Skill | Description |
|-------|-------------|
| `problem-statement` | Articulate problems from user perspective (who, what, why, how they feel) |
| `proto-persona` | Create assumption-based persona profiles from research and stakeholder input |
| `user-story` | Write user stories with Mike Cohn format + Gherkin acceptance criteria |
| `user-story-splitting` | Break down epics into smaller, independently deliverable stories |
| `user-story-mapping` | Visualize user journey as hierarchical map with narrative flow |
| `jobs-to-be-done` | Explore customer motivations (functional, social, emotional jobs + pains + gains) |
| `press-release` | Create visionary press release using Amazon's "Working Backwards" method |
| `positioning-statement` | Geoffrey Moore-style positioning (who, what need, category, benefit, difference) |
| `epic-hypothesis` | Frame epics as testable if/then hypotheses with validation approach |
| `customer-journey-map` | Visualize customer interactions across all stages (awareness to loyalty) |
| `pestel-analysis` | Analyze macro factors: Political, Economic, Social, Technological, Environmental, Legal |
| `company-research` | Create comprehensive company profiles from public sources |
| `recommendation-canvas` | Evaluate AI product solutions (business outcomes, customer outcomes, risks) |
| `pol-probe` | Create lightweight validation artifacts to test hypotheses before heavy investment |
| `storyboard` | 6-frame visual narrative of user journey from problem to solution |
| `eol-message` | Craft empathetic end-of-life messages for product/feature discontinuation |
| `saas-revenue-growth-metrics` | Calculate revenue, retention, growth metrics (MRR, ARR, churn, NRR) |
| `saas-economics-efficiency-metrics` | Evaluate unit economics (CAC, LTV, payback, Rule of 40, magic number) |
| `finance-metrics-quickref` | Fast lookup table for 32+ SaaS finance metrics with benchmarks |
| `tam-sam-som-calculator` | Calculate Total/Serviceable/Obtainable Market |

### Interactive Skills (17)

Guided flows with adaptive questions and context-aware recommendations.

| Skill | Description |
|-------|-------------|
| `discovery-interview-prep` | Prepare for customer discovery interviews with adaptive questions |
| `customer-journey-mapping-workshop` | Create journey maps through guided workshop questions |
| `user-story-mapping-workshop` | Build user story maps with two-dimensional structure |
| `opportunity-solution-tree` | Create OST by extracting outcomes, generating opportunities, mapping solutions |
| `prioritization-advisor` | Choose right prioritization framework based on product stage and context |
| `epic-breakdown-advisor` | Break down epics using Richard Lawrence's 9 splitting patterns |
| `problem-framing-canvas` | MITRE's Problem Framing Canvas with Look Inward/Outward/Reframe |
| `lean-ux-canvas` | Jeff Gothelf's Lean UX Canvas v2 for sprint-level learning |
| `positioning-workshop` | Discover and articulate product positioning through guided questions |
| `pol-probe-advisor` | Select right PoL probe based on hypothesis, risk, and resources |
| `ai-shaped-readiness-advisor` | Assess AI-first vs AI-shaped product maturity across 5 competencies |
| `context-engineering-advisor` | Diagnose context stuffing vs engineering, advise on memory architecture |
| `tam-sam-som-calculator` | Calculate market size with adaptive questions |
| `feature-investment-advisor` | Evaluate features with ROI, revenue impact, strategic value |
| `acquisition-channel-advisor` | Evaluate channels using unit economics, recommend scale/test/kill |
| `finance-based-pricing-advisor` | Analyze pricing changes with ARPU, conversion, churn, NRR impact |
| `business-health-diagnostic` | Diagnose SaaS health, identify red flags, prioritize actions |
| `workshop-facilitation` | Facilitate workshop sessions with quick-select options |

### Workflow Skills (5)

Multi-step processes orchestrating multiple skills.

| Skill | Description |
|-------|-------------|
| `discovery-process` | Complete discovery cycle from problem hypothesis to validated solution |
| `prd-development` | Structured PRD creation orchestrating problem framing, research, solution definition |
| `roadmap-planning` | Strategic roadmap planning with prioritization, epic definition, stakeholder alignment |
| `product-strategy-session` | Comprehensive strategy session orchestrating positioning, problem framing, discovery |
| `skill-authoring-workflow` | Turn raw PM content into publish-ready skills |
